Contributing to this Challenge
------------------------------------------------------------------------------------------
Thank you for helping make this exercise better! Here are the ways you can contribute:
--Found a new vulnerability in one of the codebases?

Open a Pull Request with:
-A description of the vulnerability and its line number
-A realistic exploit scenario

The fix
Please keep any confirmed vulnerabilities out of public Issues — use the Instructor Access template to discuss with the maintainer first, to avoid spoiling the exercise for students.
Built a better prompt for a level?
Open an Issue titled "Better prompt for Level X" and share:
The prompt text
What additional vulnerabilities it surfaced compared to the starter prompt
Which AI agent you used
Found a bug in the code examples?
If a code example is broken (crashes unexpectedly, doesn't run, produces wrong baseline output), please open an Issue with the error message and your Python version.
Want to add a Level 4?
Open an Issue first to discuss the proposed difficulty, vulnerability types, and scenario before submitting a PR. We want to keep levels pedagogically coherent.
---
All contributions are appreciated. Code of conduct: be constructive and respectful.
